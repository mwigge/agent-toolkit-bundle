# /index — Update <your-docs-dir>/ Index and Memory After a Work Session

Scan the AI workspace, update the master index, and record session state in memory.md.

## Steps

### 1. Scan the AI workspace

Scan `~/<your-dev-dir>/agent-toolkit-bundle/` for all artefacts:

**Skills** — find all skill directories and their SKILL.md or README files:
```bash
ls ~/<your-dev-dir>/agent-toolkit-bundle/skills/
```
For each skill directory, read its main description file to get a one-line summary.

**Agents** — list all `.md` files in `.claude/agents/`:
```bash
ls ~/<your-dev-dir>/agent-toolkit-bundle/.claude/agents/
```
For each agent file, read the `description:` field from the YAML frontmatter.

**Commands** — list all `.md` files in `.claude/commands/`:
```bash
ls ~/<your-dev-dir>/agent-toolkit-bundle/.claude/commands/
```
For each command file, read the first `#` heading to get the command name and purpose.

**Hooks** — look for any hook scripts:
```bash
find ~/<your-dev-dir>/agent-toolkit-bundle/ -name "*.sh" -o -name "settings.json" 2>/dev/null | grep -v node_modules | grep -v ".git"
find ~/<your-dev-dir>/ -maxdepth 4 -name "settings.json" -path "*/.claude/*" 2>/dev/null
```

### 2. Check existing index

Read the current index if it exists:
```
~/<your-dev-dir>/<your-docs-dir>/INDEX.md
```

Note what has changed since the last index update.

Write to `~/<your-dev-dir>/<your-docs-dir>/INDEX.md`:

```markdown

```

Populate every table with the real data collected in step 1.

### 4. Update memory.md

Read the current `~/<your-dev-dir>/agent-toolkit-bundle/memory.md`.

Update the following sections:

**Date:** Update "Session State" date to today.

**Recent changes (append, don't overwrite):**
Add a new entry at the top of the recent changes section:
```
## <YYYY-MM-DD> — /index run
- INDEX.md updated with <N> agents, <N> commands, <N> skills
- <any other notable changes from this session if known>
```

**What's complete:**
If the user mentioned completed work during this session, add it.

**What's pending:**
Preserve existing pending items; add new ones if known.

Write the updated memory.md back.

### 5. Output a summary

```
## Index updated

### <your-docs-dir>/INDEX.md
- <N> agents documented
- <N> commands documented
- <N> skills documented
- <N> hooks/settings documented

### agent-toolkit-bundle/memory.md
- Date updated to <YYYY-MM-DD>
- Recent changes entry added

Changes since last index:
- New: <list any files that are new vs the previous index>
- Removed: <list any files removed>
- Updated: <list any files that changed>
```
