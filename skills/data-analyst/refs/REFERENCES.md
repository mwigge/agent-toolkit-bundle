# Data Analyst — Reference Links

## DataFrames & In-Process Analytics
- https://pandas.pydata.org/docs/ — pandas: the standard Python DataFrame library, time series, I/O
- https://pandas.pydata.org/docs/user_guide/indexing.html — pandas indexing: loc, iloc, boolean indexing
- https://pola.rs/docs/ — Polars: fast DataFrame library written in Rust, lazy evaluation, Apache Arrow
- https://docs.pola.rs/user-guide/lazy/ — Polars lazy API: build query plans, optimize before execution

## In-Process SQL
- https://duckdb.org/docs/ — DuckDB: in-process OLAP SQL engine, reads Parquet/CSV/Arrow natively
- https://duckdb.org/docs/api/python/overview — DuckDB Python API: `duckdb.connect()`, `.sql()`, `.df()`

## Notebooks
- https://jupyter.org/documentation — Jupyter documentation: notebooks, kernels, magic commands
- https://nbformat.readthedocs.io/en/latest/ — nbformat: notebook file format (.ipynb JSON structure)
- https://docs.jupyter.org/en/latest/running.html — Running notebooks: papermill, nbconvert, voila

## Data Quality
- https://docs.greatexpectations.io/docs/ — Great Expectations: data validation framework, expectations, data docs
- https://docs.greatexpectations.io/docs/core/introduction/gx_overview — GX overview: expectations, suites, checkpoints

## Best Practices
- https://www.tidydata.org/ — Tidy Data (Wickham): principles for structured data analysis
