# Security Review — Reference Links

## OWASP
- https://owasp.org/www-project-top-ten/ — OWASP Top 10 (2021): A01–A10 web application security risks
- https://genai.owasp.org/resource/owasp-top-10-for-llm-applications-2025/ — OWASP Top 10 for LLM Applications / MCP 2025: prompt injection, insecure tool calls, data leakage
- https://owasp.org/www-project-application-security-verification-standard/ — OWASP ASVS: detailed security requirements checklist (Level 1/2/3)
- https://cheatsheetseries.owasp.org/ — OWASP Cheat Sheet Series: XSS, SQL Injection, CSRF, authentication, etc.

- https://pypi.org/project/pip-audit/ — pip-audit: scan Python dependencies for known CVEs

- https://docs.npmjs.com/cli/v10/commands/npm-audit — npm audit: dependency vulnerability scanning
