# Statistical Analysis — Reference Links

## Core Libraries
- https://docs.scipy.org/doc/scipy/reference/stats.html — scipy.stats: hypothesis tests, distributions, descriptive stats
- https://www.statsmodels.org/stable/index.html — statsmodels: regression, time series, statistical tests
- https://pingouin-stats.org/ — pingouin: easy-to-use statistical library (ANOVA, correlation, power analysis, effect sizes)
- https://scikit-learn.org/stable/modules/preprocessing.html — scikit-learn preprocessing: StandardScaler, normalization, encoding

## Specific Tests & Methods
- https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.shapiro.html — Shapiro-Wilk normality test (n < 50)
- https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.kstest.html — Kolmogorov-Smirnov test (n ≥ 50)
- https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.ttest_ind.html — Two-sample independent t-test
- https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.mannwhitneyu.html — Mann-Whitney U test (non-parametric alternative)
- https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.bootstrap.html — Bootstrap confidence intervals (scipy 1.7+)

## Effect Sizes
- https://pingouin-stats.org/build/html/generated/pingouin.compute_effsize.html — pingouin effect sizes: Cohen's d, Hedges g, eta-squared, odds ratio
- https://en.wikipedia.org/wiki/Cohen%27s_d — Cohen's d: small (0.2), medium (0.5), large (0.8) effect size thresholds

## Best Practices
- https://www.nature.com/articles/nmeth.3288 — Points of Significance series (Nature Methods): statistical fundamentals for scientists
- https://statisticsbyjim.com/hypothesis-testing/ — Statistics by Jim: practical hypothesis testing guides
