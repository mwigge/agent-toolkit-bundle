# Presentation & Technical Communication — References

## Frameworks

- **Diátaxis** — The canonical framework for structuring technical documentation
  https://diataxis.fr/

- **C4 Model** — Simon Brown's layered architecture diagram approach (Context, Container, Component, Code)
  https://c4model.com/

## Diagramming Tools

- **Mermaid** — Text-as-code diagrams that render in GitHub, GitLab, Notion, and most wikis
  https://mermaid.js.org/

- **PlantUML** — Older but richer text-as-code diagrams; C4-PlantUML library available
  https://plantuml.com/

## Presentation Design

- **Assertion-Evidence Slide Design** — Michael Alley's research-backed approach to technical slides
  https://www.assertion-evidence.com/

## Technical Writing

- **Google Technical Writing Courses** — Free, practical courses on clarity, active voice, and sentence structure
  https://developers.google.com/tech-writing

## Additional Reading

- *Presentation Patterns* — Neal Ford, Matthew McCullough, Nathaniel Schutta — patterns for conference talks
- *The Visual Display of Quantitative Information* — Edward Tufte — foundational data visualisation principles
- WCAG 2.1 Quick Reference (accessibility) — https://www.w3.org/WAI/WCAG21/quickref/
