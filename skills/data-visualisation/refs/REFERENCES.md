# Data Visualisation — Reference Links

## Static Plotting
- https://matplotlib.org/stable/contents.html — Matplotlib: comprehensive 2D plotting library, publication-quality figures
- https://matplotlib.org/stable/gallery/index.html — Matplotlib gallery: examples for every chart type
- https://seaborn.pydata.org/ — Seaborn: statistical data visualisation built on Matplotlib
- https://seaborn.pydata.org/api.html — Seaborn API reference: heatmap, pairplot, violinplot, etc.

## Interactive Plotting
- https://plotly.com/python/ — Plotly Python: interactive, web-embeddable charts (scatter, line, bar, maps)
- https://plotly.com/python/plotly-express/ — Plotly Express: high-level API for common chart types
- https://altair-viz.github.io/ — Vega-Altair: declarative visualisation using Vega-Lite grammar

## Web-Native
- https://observablehq.com/plot/ — Observable Plot: JavaScript charting for the modern web (used in Observable notebooks)

## Accessibility & Colour
- https://colorbrewer2.org/ — ColorBrewer: perceptually safe, print-friendly colour palettes for maps and charts
- https://seaborn.pydata.org/tutorial/color_palettes.html — Seaborn color palettes: sequential, diverging, qualitative

## Best Practices
- https://www.edwardtufte.com/book/the-visual-display-of-quantitative-information/ — The Visual Display of Quantitative Information (Tufte): data-ink ratio, chartjunk avoidance
- https://clauswilke.com/dataviz/ — Fundamentals of Data Visualization (Wilke): free online book, accessibility, colour
